blog 安装的Node.js插件越来越多，记录下  :grin:
```shell
cnpm install hexo-git-backup --save  //备份插件
cnpm install hexo-neat --save  //静态资源压缩插件
cnpm install hexo-abbrlink --save  //url格式化插件
cnpm install hexo-wordcount --save  //文字统计插件
cnpm install hexo-generator-feed //rss
--- 替换渲染引擎
cnpm un hexo-renderer-marked --save
cnpm install hexo-renderer-markdown-it --save
cnpm install markdown-it-emoji --save

```
